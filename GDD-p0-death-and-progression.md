# P0-2：失败条件与局外成长 — 详细设计

> 版本：v0.1  
> 依据决策：E=E1（永久死亡 + 特殊遗物复活）、F=解锁 + 属性加点

---

## 一、死亡系统

### 1.1 核心规则（决策E：永久死亡）

**HP ≤ 0 → Run 结束。**

- 所有局内进度消失（笔力、金币、词块库、技能、遗物）
- 回到主菜单
- 弹出 Run 总结面板（见第四节）
- 根据 Run 表现获得局外成长资源

### 1.2 唯一例外：复活遗物

**遗物「不死鸟之羽」**（传说级）：

| 属性 | 内容 |
|------|------|
| 图标 | 🪶 |
| 名称 | 不死鸟之羽 |
| 稀有度 | 传说 |
| 效果 | HP 归零时自动触发：HP 回复至 50%，此遗物销毁 |
| 获取途径 | Boss 遗物选择（3选1中可能出现）、极稀有事件 |
| 设计限制 | 不可叠加（同一种遗物不重复获取） |

触发效果：
```
HP 归零 → 屏幕碎裂效果 → 凤凰羽毛燃烧动画 → HP 回复至 50% → 
日志：「不死鸟之羽燃尽了……你从灰烬中站起。」
→ 战斗继续（不回到地图，同一场战斗继续进行）
```

### 1.3 复活后状态

| 项目 | 状态 |
|------|------|
| HP | 最大 HP 的 50% |
| 笔力 | 保留死亡时的数值 |
| 压力 | 保留死亡时的数值 +3（涅槃的代价） |
| 法力 | 回满（相当于新回合开始） |
| 敌人 | 不重置，保持死亡时的状态 |
| 词块库 | 保留 |
| 遗物 | 不死鸟之羽销毁，其他保留 |
| 技能 | 保留 |

## 二、局外成长系统（决策F：解锁 + 加点）

分两个独立模块：

```
局外成长
├── 模块A: 里程碑解锁（不需要货币，达到条件即解锁）
│   ├── 新词块进入随机池
│   ├── 新遗物进入随机池
│   ├── 新技能进入技能池
│   ├── 新事件进入事件池
│   └── 新角色/起始预设
│
└── 模块B: 灵感升级（消耗"灵感"货币，加点增强属性）
    ├── 最大 HP
    ├── 起始笔力
    ├── 起始法力上限
    ├── 法力回复量
    ├── 基础攻击力
    └── 特殊能力
```

## 三、模块A：里程碑解锁

### 3.1 解锁结构

```
里程碑 = { 条件, 解锁内容 }

条件类型：
  - 首次到达第 N 层
  - 首次击败某敌人类型
  - 累计击败某敌人 N 次
  - 首次触发某 combo
  - 首次通关（击败 Boss）
  - 使用某角色通关
  - 单局完成 N 次交换
```

#### 词块解锁

| 首次通关 | "传说中的" | 形容词 |

## 四、模块B：灵感升级

### 4.1 灵感点获取

每次 Run 结束后，根据表现获得"灵感"点数：

```
灵感 = 基础分 + 层数分 + 战斗分 + 首次奖励

基础分：完成一场战斗 = 1 点（失败也计）
层数分：到达的最深层数 × 2
战斗分：每场精英战斗 +2，Boss 每打空一管血 +3
首次奖励：（每周目第一次触发时）
  - 首次通关 +20
  - 首次触发 combo +5
  - 首次击败 Boss +10
  - 首次使用某角色通关 +15

最终灵感 = 上述总和 × 难度倍率
  难度1 = ×1.0
  难度2 = ×1.3
  难度3 = ×1.6
  ...
```

### 4.2 升级项目与花费

| 升级项目 | 效果 | 等级1 | 等级2 | 等级3 | 等级4 | 等级5 |
|----------|------|-------|-------|-------|-------|-------|
| 体魄 | 最大 HP +3 | 5✨ | 10✨ | 20✨ | 35✨ | 55✨ |
| 笔锋 | 起始笔力 +1 | 10✨ | 25✨ | 50✨ | — | — |
| 专注 | 起始法力上限 +1 | 10✨ | 25✨ | 50✨ | — | — |
| 回响 | 每回合法力回复 +1 | 10✨ | 25✨ | 50✨ | — | — |
| 锋芒 | 基础攻击力 +1 | 5✨ | 10✨ | 20✨ | 35✨ | 55✨ |
| 韧性 | 起始 HP +5 | 10✨ | 25✨ | 50✨ | — | — |

**设计说明**：
- 等级越高越贵，让玩家在多个项目间分散投资
- 笔力是最强资源，升到满也只有 +3 起始笔力（加上基础 3 = 6）
- 法力上限和回复有上限，不会无限提升
- 所有升级在同一棵树上，玩家看不到全部等级（渐进式展示）

## 六、设计总结

| 系统 | 作用 | 给玩家的感觉 |
|------|------|-------------|
| 永久死亡 | 确立风险基线 | "每场战斗都要认真对待" |
| 不死鸟之羽 | 稀有安全网 | "这次 run 运气好，有退路" |
| 里程碑解锁 | 探索驱动力 | "差一点就到第 6 层了，再试一次" |
| 灵感加点 | 失败补偿 + 目标感 | "死了也拿了 15 点，再来一局就能升级了" |
| Run 总结面板 | 仪式感和成就感 | "我这次比上次多走了 3 层" |

两个模块的节奏配合：
- **前期**（0-5 次 run）：大量里程碑解锁，每次 run 都有新东西
- **中期**（5-20 次 run）：解锁放缓，灵感加点成为主要驱动
- **后期**（20+ 次 run）：角色/高难度解锁，挑战自我
