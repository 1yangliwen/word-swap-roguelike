# P0-1：战后奖励流程 — 详细设计

> 版本：v0.1  
> 依据决策：A=A2（随机3选1技能）、B=B1（遗物3选1）、C=固定掉落（金币+词块）

---

## 一、战后结算流程

```
战斗结束 → [阶段1: 总结语] → [阶段2: 基础奖励] → [阶段3: 战利品] → 回到地图
```

三个阶段在一个面板上依次呈现，玩家手动推进。

---

## 二、阶段1：战斗总结语（0.5秒自动生成）

### 2.1 数据来源

| 提取要素 | 来源 | 决定总结语中的… |
|----------|------|----------------|
| 最后一击方式 | 造成击杀的操作类型 | 动词词块 |
| 敌人名 | 本场敌人 | 名词词块 |
| 残留环境状态 | 本场最后存在的环境/场地效果 | 形容词词块 |
| 关键数字 | max(玩家剩余HP, 本场最大单次伤害) | 数字词块 |

### 2.2 动词映射表

| 最后一击来源 | 动词 | 示例 |
|-------------|------|------|
| 劈砍技能 | "劈砍" | 「你用**劈砍**了…」 |
| 火焰技能 | "焚尽" | 「你用**焚尽**了…」 |
| 冰冻技能 | "冻结" | 「你用**冻结**了…」 |
| 雷电技能 | "轰杀" | 「你用**轰杀**了…」 |
| 交换主语（敌人自伤） | "逆转" | 「你用**逆转**了…」 |
| 场地燃烧 DOT | "烧灼" | 「你用**烧灼**了…」 |
| 反击 | "反噬" | 「你用**反噬**了…」 |
| 默认（普攻收尾） | "击败" | 「你用**击败**了…」 |

### 2.3 形容词映射表

| 本场最终状态 | 形容词 |
|-------------|--------|
| 场地处于燃烧 | "燃烧的" |
| 场地处于冻结 | "冰封的" |
| 敌人身上有 buff 词块 | 取该词块的值（如"坚固的"） |
| 玩家武器形容词（无特殊场地） | "沾满血的" + 武器形容词（拼接） |
| 无特殊情况 | "疲惫的" |

### 2.4 生成示例

同一场战斗不同打法产生不同总结语：

```
打法A：用火焰斩烧死哥布林，场地燃烧，剩 8 HP
→ 「你用 焚尽 了 哥布林 ，留下了 燃烧的 伤痕和 8点 余烬」

打法B：交换主语让哥布林自己打死自己，剩 5 HP  
→ 「你用 逆转 了 哥布林 ，留下了 愚蠢的 传说和 5点 余烬」
```

### 2.5 总结语的生命周期

- 总结语中的 4 个词块（动词、名字、形容词、数字）注册到 token pool（zone="summary"）
- 在**下一场完整的战斗**中作为常驻可交换素材存在
- 下一场战斗结束后，旧总结语被新的替换
- 如果下一场是事件/休息（非战斗节点），总结语保留到再下一场战斗

## 三、阶段2：基础奖励（自动结算）

```
奖励面板：
┌─────────────────────────────────────────┐
│  « 战斗胜利 »                            │
│                                         │
│  💰 金币 +{amount}   （当前: X）         │
│  ✒️ 笔力 +1           （当前: X → X+1）   │
│  ✦ 法力回满           （当前: 满）        │
│                                         │
│  [继续 →]                               │
└─────────────────────────────────────────┘
```

### 3.1 金币掉落表

| 敌人类型 | 金币 |
|----------|------|
| 普通敌人 | 8-12 |
| 精英敌人 | 20-30 |
| Boss | 40-60 |

### 3.2 笔力奖励

| 战斗类型 | 笔力 |
|----------|------|
| 普通战斗 | +1 |
| 精英战斗 | +1 |
| Boss | +2 |

### 3.3 法力规则（决策D：像杀戮尖塔）

- **每场战斗开始时**法力回满到最大值
- 战斗中每回合自动回复 3 点法力
- 战后不额外回复（因为下场战斗开始时会回满）
- 设计意义：法力是"单场战斗内"的资源（类似尖塔的能量），笔力是"跨战斗"的资源（类似尖塔的血量）

## 四、阶段3：战利品（手动操作）

### 4.1 普通战斗战利品（决策C：固定掉落）

**每个敌人有固定的战利品掉落表**，不随机抽取，直接给。

| 敌人 | 掉落词块 | 类型 |
|------|---------|------|
| 哥布林 | "生锈的" | 形容词 |
| 毒蛇 | "剧毒的" | 形容词 |
| 石壳虫 | "坚固的" | 形容词 |
| 盗贼 | "锋利的" | 形容词 |
| 巫师 | "燃烧的" | 形容词 |
| 铁甲骑士 | "坚固的" | 形容词 |
| 咒术师 | "诅咒的" | 形容词 |
| 噬词兽 | "扭曲的" | 形容词 |

界面：

```
┌─────────────────────────────────────────┐
│  战利品                                  │
│                                         │
│  哥布林留下了：                          │
│  ┌──────────┐                           │
│  │ 生锈的    │  ← 点击加入词块库         │
│  │ [形容词]  │                           │
│  └──────────┘                           │
│                                         │
│  也可以选择：                            │
│  [不要这个词块，换成 +1 笔力]            │
│                                         │
└─────────────────────────────────────────┘
```

**关键交互（延续 GDD 11.3 设计）**：
- 战利品面板上的词块是**可交换的**——玩家可以花笔力把战利品词块和身上已有的词块交换
- 交换后再决定"拿走"还是"换笔力"
- 面板上同时显示当前武器形容词和词块库中的同类型词块，方便玩家决策

### 4.2 精英战斗战利品（决策A：随机3选1技能）

```
┌─────────────────────────────────────────┐
│  技能奖励                                │
│                                         │
│  选择一项技能加入你的技能槽：             │
│                                         │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ │
│  │ 火焰斩    │ │ 冰冻斩    │ │ 威吓      │ │
│  │ ✦3 输出  │ │ ✦4 输出  │ │ ✦2 操控  │ │
│  │ 造成3伤+  │ │ 造成4伤+  │ │ 敌人转为  │ │
│  │ 下回合2伤 │ │ 减速1回合 │ │ 防御姿态  │ │
│  └──────────┘ └──────────┘ └──────────┘ │
│                                         │
│  (4个技能槽已有 [劈砍][格挡][空][空])    │
│  选择后若槽满，必须替换一个现有技能       │
│                                         │
└─────────────────────────────────────────┘
```

**技能池分层**（配合地图进度）：

| 地图阶段 | 可选技能池 |
|----------|-----------|
| 第 1-3 层 | 基础技能池（劈砍、格挡、威吓、石肤、火焰斩） |
| 第 4-8 层 | 扩展技能池（基础 + 雷霆一击、反击姿态、暴露弱点、解锁、冰冻斩） |
| 第 9-12 层 | 完整技能池（扩展 + 暗影步、时间扭曲、镜像术等稀有技能） |

**技能槽已满时的处理：**
- 选新技能 → 弹出 4 选 1 替换（选哪个被替换掉）
- 也可以取消，不拿新技能

### 4.3 Boss 战利品（决策B：遗物3选1，全传说）

Boss 掉落的遗物**全部是传说级**。3 选 1，每个都直接改变交换规则或战斗底层逻辑。

```
┌─────────────────────────────────────────┐
│  传说收藏品                              │
│                                         │
│  选择一件传说收藏品：                     │
│                                         │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ │
│  │ 🏺 无尽墨 │ │ 🪶 镜像羽 │ │ 🪶 不死鸟 │ │
│  │ [传说]   │ │ [传说]   │ │ [传说]   │ │
│  │ 30%概率  │ │ 交换时复 │ │ HP归零时 │ │
│  │ 不耗笔力  │ │ 制词块   │ │ 复活一次  │ │
│  └──────────┘ └──────────┘ └──────────┘ │
│                                         │
└─────────────────────────────────────────┘
```

**Boss 遗物抽取规则：**
- 从传说池中随机抽 3 个（排除已拥有的）
- 传说遗物**只**从 Boss 奖励或极稀有事件中获得
- 精英和商店不出传说

## 六、金币系统

### 6.1 金币的用途（在商店中使用，商店设计见后续）

| 用途 | 价格 |
|------|------|
| 购买词块（随机品质） | 6-10 金币 |
| 购买技能（随机 2 选 1） | 12-20 金币 |
| 购买遗物（普通/稀有） | 15-25 金币 |
| 移除词块库中一个词块 | 5 金币 |
| 回复 HP | 2 金币/HP |
| 净化压力 | 4 金币/点 |

### 6.2 金币规则

- 金币跨战斗继承、跨层继承
- 金币不可被交换（不属于文字系统的一部分，是纯粹的经济货币）
- 不在 UI 上显示为 token（和 HP/笔力不同）
- 上限 999

## 七、法力规则（决策D：像杀戮尖塔）

### 7.1 法力生命周期

```
战斗开始 → 法力回满（10/10）
回合开始 → 法力回复 +3（不超过上限）
使用技能 → 消耗法力
回合结束 → 法力保留
下一回合 → 法力 +3
...
战斗结束 → （法力状态不重要了，下战开始会回满）
```

### 7.2 法力上限

- 初始最大值：10
- 可通过遗物/事件永久提升最大值（类似尖塔的 +1 能量遗物）

### 7.3 设计差异总结

| 资源 | 跨战斗继承 | 类比 |
|------|-----------|------|
| **法力** | 不继承（每战回满） | 尖塔的能量 |
| **笔力** | 继承（稀缺资源） | 类似尖塔的血量 |
| **压力** | 继承（累积代价） | 独有系统 |
| **HP** | 继承 | 尖塔的血量 |
| **金币** | 继承 | 尖塔的金币 |
| **词块库** | 继承（但可丢弃） | 独有系统 |
